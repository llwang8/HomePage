Pomodoro Clock

I built an animated Pomodoro Clock with JavaScript, jQuery, HTML, CSS, Bootstrap and liquidBubble plugin. User can start a default 25 minute pomodoro, and the timer will go off once 25 minutes has elapsed and reset itself to break setting. User can change the setting of the clock for the next pomodoro and break by clicking "-" and "+". User can click the clock face to start / pause the clock.
